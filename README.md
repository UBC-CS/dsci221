# DSCI 221: Data Structures for Data Science

Course website for DSCI 221 at the University of British Columbia.

## About

This course covers fundamental data structures and algorithms with a focus on applications in data science. Topics include asymptotic analysis, recursion, stacks, queues, dictionaries, trees, heaps, and graphs.

## Website

The course website is available at: https://ubc-cs.github.io/dsci221/

## Development

The site is built using [Quarto](https://quarto.org/). To preview locally:

```bash
quarto preview
```
